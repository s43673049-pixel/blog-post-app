What is HTML?

HTML (HyperText Markup Language) is the foundation of every website. It is used to structure content on the web such as text, images, links, forms, and more.

If you are starting your journey in web development, learning HTML is your first and most important step.

What Does HTML Do?

HTML tells the browser:

  • What content to display
  • How content is structured
  • Where headings, paragraphs, images, and links go

HTML does not style or add logic — that’s done by CSS and JavaScript.

Basic HTML Structure

Every HTML page follows a standard structure:


<!DOCTYPE html>
<html>
<head>
  <title>My First Website</title>
</head>
<body>

  <h1>Hello World</h1>
  <p>This is my first HTML page.</p>

</body>
</html>

Explanation:

  • <!DOCTYPE html> → tells browser it's HTML5
  • <html> → root element
  • <head> → metadata (title, styles)
  • <body> → visible content

Common HTML Tags

Tag Description
<h1> to <h6> Headings
<p> Paragraph
<a> Hyperlink
<img> Image
<ul>, <ol>, <li> Lists (unordered, ordered, list item)
<div> Container
<span> Inline Container

Why HTML is Important

  • Works on all browsers
  • Easy to learn
  • Forms the backbone of the web
  • Required for all frontend frameworks

Conclusion

HTML is simple but powerful. Mastering HTML gives you a strong base for CSS, JavaScript, and frameworks like React.
