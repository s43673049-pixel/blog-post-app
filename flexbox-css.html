Introduction

Flexbox is a modern CSS layout system that helps you align elements easily in rows or columns.

Before Flexbox, layouts were difficult and messy. Flexbox makes responsive design simple and clean.

Flexbox Container

To use Flexbox, apply display: flex to a parent element:


  .container {
  display: flex;
}

Example Layout


  <div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>


  .container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.box {
  width: 100px;
  height: 100px;
  background: #4f46e5;
  color: white;
}

Important Flexbox Properties

justify-content (horizontal)

  • flex-start (default)
  • flex-end
  • center
  • space-between
  • space-around

align-items (vertical)

  • stretch (default)
  • flex-start
  • flex-end
  • center
  • baseline

Why Use Flexbox?

  • Easy alignment
  • Responsive design
  • Clean code
  • Supported by all modern browsers
